Részletes leírás

A robot szimulátor.

Van egy belső, RobotState típusú állapota, melyet egy QTimer segítségével periodikusan frissít. Létrehoz egy CommunicationTcpSocketServer objektumot a kommunikációhoz, amihez lehet csatlakozni. Minden szimulációs lépés után elküldi az állapotát. Ha egy RobotState objektumot kap, azt parancsként értelmezi.
