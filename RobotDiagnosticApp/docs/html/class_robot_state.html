Részletes leírás

A robot teljes állapotleírása le egy adott időpillanatban.

A kommunikáció során ilyen objektumokat küldünk át a robot és a kliens között. A robottól érkezve az aktuális állapotot képviseli, míg az ellenkező irányba küldve parancsként a kívánt állapotot írja le.

A history ilyen objektumok sorozata.

Enumeráció-tagok dokumentációja

enum RobotState::Status
strong

A robot állapota.

Enumeráció-értékek
Default 

Alapállapot. A robot tartja az aktuális gyorsulást.

Reset 

Reset. Parancsként használjuk.

Stopping 

Stop parancs. Ez az állapot akkor is érvényes marad, ha megállt a robot.

Accelerate 

Gyorsítási parancs a robotnak.

Turn 

Kanyarodás parancs. A fordulás mértékét kívülről kapja.

SelfTest 

Öntesztelő állapot
